Date
2019/05/31 Linked list library with basic support routines for reading 1 to N valid pixels in a satellite image and producing an NCODA QC compliant output.

Added log scale capability to optics due to NCODA downstream issues with the spectral shape of optics by transforming the data to a more bell shaped curve we avoid non-physical values (negatives) and keep optics more like SST thus enabling more stock NCODA* utilization. Values will have to be reverted back once in the model prior to use. This ensures less code modification and keeps everything consistent within NCODA* proper.
