Detailed Description

Contains definitions for a Product structure necessary to hold salient meta-data related to a scientific data point. Information is gathered from the NetCDF/HDF5 file for persistence thus ensuring is does have to continually be accessed for the same data.

Author
NRL, Code 7330
Date
2019/05/31 Library used to store information about a product gathered from the parent NetCDF/HDF5 file and persisted to support optics data processing as the satellite file is parsed into individual slices.
